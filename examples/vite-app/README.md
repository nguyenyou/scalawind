# Scalawind Vite

## Install dependencies

```bash
$ npm install
# or
$ bun install
# or
$ pnpm install
# or
$ yarn install
```

## Start dev server

```bash
$ npm run dev:scala
# or
$ bun dev:scala
# or
$ pnpm dev:scala
# or
$ yarn dev:scala
```

In another terminal:

```bash
$ npm run dev
# or
$ bun dev
# or
$ pnpm dev
# or
$ yarn dev
```

Open: http://localhost:5173
